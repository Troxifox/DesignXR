# XR Toon Tutor

XR Toon Tutor is an immersive classroom prototype. Learners explore a 3D-style character scene, switch between design lessons, hear voiceover explanations, and answer short activities.

## Project Details

- Project Title: XR Toon Tutor
- Problem Statement: Students often struggle to understand design concepts like scale, color, lighting, and spatial relationships from flat slides or static images.
- Solution Overview: XR Toon Tutor turns a lesson into an immersive scene with a character guide, interactive controls, voiceover, and quick activities.
- Target Users: Students, teachers, design beginners, remote classrooms, and XR learning labs.
- Platform Details: Desktop browser prototype with mobile and headset presentation in mind.
- Tech Stack: HTML, CSS, JavaScript, Canvas 3D simulation, Web Speech API.

## Walkthrough Flow

1. Introduce the problem.
2. Show the immersive classroom scene.
3. Start Scale & Distance, Color Theory, and Lighting & Shadows.
4. Turn on voiceover and show the guide text.
5. Answer activity questions and show the score.
6. Click Focus Mode and explain the headset/mobile direction.
7. End with the learning impact.
